Cos'è un Memory Leak nell'Heap?

Un memory leak nell'heap si verifica quando un programma alloca memoria (come mostrato dai quadratini rossi) ma non la libera quando non è più necessaria. Nel tempo, questi blocchi di memoria "persi" si accumulano, riducendo la memoria disponibile e potenzialmente causando il crash del programma.

Simulazione: Clicca sui quadratini per allocare memoria, poi usa il pulsante "Simula Memory Leak" per mostrare cosa succede quando questa memoria non viene deallocata.
